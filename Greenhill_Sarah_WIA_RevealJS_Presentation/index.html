For this presentation, I decided to focus on a plugin for jQuery called Tubular. In a past class, I had seen an example of this done through the use of HTML5 and CSS3 and thought that it was a really cool concept.

What is Tubular?

Tubular is a plugin that works in conjunction with jQuery in order to allow the entire background of a web page to become a YouTube video. It uses the z-index to place the footage behind the page's content, ensuring that the user will not be able to accidentally click on the video itself or navigate away from the page.

Why Tubular?

The reason that I chose to use Tubular as my focus is because, firstly, that I just plain think it's an awesome concept. Secondly, it's good to have commonality amongst sites because it creates a positive and familiar user experience, but I think that setting your site apart visually can possibly gain you more traffic purely because it's something the user hasn't seen before, but doesn't hinder their ability to use the site in general.

What are the benefits to using Tubular?

  • This is a new way to create an eye-catching design for any site.
  • Keeping the user interested long enough to experience your site's content is important.
  • This plugin is open source and able to be tweaked to suit your needs.
  • There are good ways to use this and bad, the difference is knowing when.
